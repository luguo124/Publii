macOS 10.15.4 系统 / 更新系统后「意外退出」及「崩溃闪退」问题修复方法

 
最近有部分网友反映更新系统后有很多软件打不开，或者出现闪退的情况，其实是因为Apple苹果公司在新系统中删除了TNT的证书。

1、打开终端工具输入并执行如下命令：

sudo codesign --force --deep --sign - (应用路径)
应用路径：打开访达（Finder），点击左侧导航栏的 应用程序，找到相关应用，将它拖进终端命令-的后面，然后按下回车即可，注意最后一个 - 后面有一个空格。
 
 
 
 
正常情况下只有一行提示，即成功：
/文件位置 : replacing existing signature
 

2、如遇如下错误（一般不会出现）：

/文件位置 : replacing existing signature
/文件位置 : resource fork,Finder information,or similar detritus not allowed
需要安装开发工具xcode，你可以在 App Store 中下载安装，安装完运行一下。（Xcode很大，8G多，你可以先按上面操作执行，如果没办法安装再来安装Xcode。）
需要安装Command Line Tools 工具，打开终端工具输入命令：xcode-select --install（弹出安装窗口后选择继续安装，安装过程需要几分钟，请耐心等待。）
写在最后 如果以上步骤执行完成后还是无法运行，尝试关闭一下SIP，（关闭SIP教程） 如果还是不行那就无法在 10.15.4 版本中使用了，稍安勿躁请关注我，等待软件的新版更新！
